Для того чтобы ваши примеры программ на языке PascalABC.NET корректно работали и соответствовали требованиям практики, они должны демонстрировать следующие ключевые этапы работы с данными:

* **Загрузка датасета**: программа должна успешно считывать CSV-файл в рабочую среду.
* **Первичный анализ**: демонстрация вывода первых строк датасета и его общей схемы (структуры данных).
* **Работа с признаками**: использование признаков и целевой переменной (target) в рамках задачи машинного обучения.
* **Базовая модель**: построение и демонстрация простейшей модели машинного обучения на основе загруженных данных.
* **Предобработка (для raw-версий)**: если вы используете версию данных с необработанными значениями (raw), пример должен содержать код базовой предобработки.

### Пример структуры кода

Ориентировочный шаблон загрузки данных в PascalABC.NET ML выглядит следующим образом:

```pascal
uses MLABC;

begin
  // Установка языка для сообщений библиотеки
  Datasets.Language := 'ru';
  
  // Загрузка конкретного датасета
  var ds: Datasets.UsedCarsPrice;
  
  // Вывод общей информации и первых строк данных
  ds.Info;
  ds.Data.Print(20);
end.

```

**Важное требование:** все примеры должны быть полностью рабочими и запускаться без необходимости ручной переработки структуры данных пользователем. Если API загрузки в актуальной версии библиотеки отличается от приведенного в материалах, пример необходимо адаптировать.
