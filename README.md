# Датасеты для ML библиотеки PascalABC.NET

## Структура датасета:
```
some_dataset/
├── data/
│   └── some_dataset.csv
├── meta/
│   └── some_dataset.meta
├── samples/
│   ├── 01_load_dataset.pas
│   ├── 02_describe_dataset.pas
│   └── 03_baseline_model.pas
└── report/
    ├── README.md
    ├── quality_report.md
    ├── ai_log.md
    └── sources.md
```
*Более подробная структура каждого файла описана в соответствующих файлах в [requirements/](requirements/)*
