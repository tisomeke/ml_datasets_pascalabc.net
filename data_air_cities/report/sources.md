# Источники данных

## data_air_cities_raw.csv

- **Тип источника:** Реальные открытые данные
- **Персональные данные:** Отсутствуют
- **Описание:** Набор данных о состоянии загрязнения атмосферы в городах России за 2007–2022 гг. Собран и обработан проектом «Если быть точным» на основе ежегодников Росгидромета.

### Источники

| Источник | Что использовано | Дата обращения |
|----------|-----------------|----------------|
| [Росгидромет — Ежегодники состояния загрязнения атмосферы](https://www.meteorf.gov.ru/) | Первичные данные измерений | 09.07.2026 |
| [GitHub: tochno-st/air_quality_cities](https://github.com/tochno-st/air_quality_cities.git) | Обработанная таблица | 09.07.2026 |
| [Проект «Если быть точным»](https://tochno.st/) | Методология обработки | 09.07.2026 |

### Лицензия
Open Data. Данные Росгидромета являются общедоступными. Обработка проекта «Если быть точным» распространяется на условиях открытой лицензии.
