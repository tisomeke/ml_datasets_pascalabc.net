# Датасет: Air Cities (Качество атмосферного воздуха в городах РФ)

**Статус:** WIP — черновик

## Цель
Анализ и прогнозирование уровня загрязнения атмосферного воздуха в городах России на основе данных Росгидромета (2007–2022 гг.). Датасет поддерживает три задачи машинного обучения: регрессию, классификацию и кластеризацию.

## Предметная область
Экология, качество атмосферного воздуха, промышленные выбросы.

## Источник данных
- **Первоисточник:** Ежегодник «Состояние загрязнения атмосферы в городах на территории России» Росгидромета (ГГО им. А.И. Воейкова)
- **Оператор:** Проект «Если быть точным» (благотворительный фонд «Нужна помощь»)
- **Репозиторий:** [github.com/tochno-st/air_quality_cities.git](https://github.com/tochno-st/air_quality_cities.git)

## Признаки

| Признак | Тип | Роль | Смысл |
|---------|-----|------|-------|
| year | Int | Признак | Год наблюдения (2007–2022) |
| region | String | Признак | Субъект РФ |
| region_oktmo | String | Признак | Код ОКТМО региона |
| city | String | Признак | Город |
| city_oktmo | String | Признак | Код ОКТМО города |
| air_general_level | Int | **Цель** | Уровень загрязнения (1–4) |
| air_standard_index | String | Признак | Вещества с СИ > 10 ПДК |
| air_repeatability | String | Признак | Наибольшая повторяемость превышения ПДК |
| air_qcp | String | Признак | Вещества с концентрацией > 1 ПДК |
| air_solid_emissions | Float | Признак | Выбросы твёрдых веществ, тыс. т |
| air_so_emissions | Float | Признак | Выбросы SO₂, тыс. т |
| air_no_emissions | Float | Признак | Выбросы NO₂, тыс. т |
| air_co_emissions | Float | Признак | Выбросы CO, тыс. т |
| air_population | Float | Признак | Численность населения, тыс. чел. |
| air_stantions | String | Признак | Количество станций мониторинга |

## Целевая переменная
`air_general_level` — общий уровень загрязнения воздуха:
- `1` — низкий
- `2` — повышенный
- `3` — высокий
- `4` — очень высокий

## Задачи ML

| Задача | Целевая переменная | Модели MLABC |
|--------|-------------------|--------------|
| **A. Регрессия** | `air_co_emissions` (выбросы CO) | `KNNRegressor` |
| **B. Классификация** | `air_general_level` (уровень 1–4) | `KNNClassifier`, `LogisticRegression` |
| **C. Кластеризация** | Нет (обучение без учителя) | `KMeans`, `DBSCAN` |

## Особенности данных
- 3992 записи, 14 признаков + целевая
- Данные реальные, собраны Росгидрометом за 2007–2022 гг.
- Множественные пропуски (NA, -1.0, «No substance», «No information»)
- Скрытые пропуски в числовых столбцах (значение -1.0)
- Текстовые списки веществ (требуют парсинга)
- Изменения СанПиН в 2014 и 2021 гг. — временной структурный сдвиг

## Использование
1. Откройте любой `.pas`-файл из папки `samples/` в среде PascalABC.NET.
2. Нажмите **F5** (Запуск) — программа загрузит `../data/data_air_cities_raw.csv`.
3. Для моделей потребуется библиотека `MLABC`.
