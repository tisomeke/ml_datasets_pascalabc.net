# Лог использования ИИ

**Дата:** 7 июля 2026 г.

## Инструменты

| Инструмент | Версия | Назначение |
|---|---|---|
| Gemini Web | 2026 | Выбор и обоснование темы датасета |
| DeepSeek Web | 2026 | Подбор коэффициентов формулы ценообразования на основе анализа реальных тарифов |
| Yandex Sourcecraft (Roo Code) | 2026 | Генерация синтетических данных, написание примеров программ на PascalABC.NET, подготовка meta-файла и отчётной документации |

## Цели использования

1. **Выбор темы датасета** — через Gemini Web предложена и обоснована тема «Тарифы мобильной связи» как подходящая для задачи регрессии.
2. **Генерация данных** — с помощью Yandex Sourcecraft сгенерирован CSV-файл (100 строк, 6 признаков + целевая переменная `price_rub`) на основе формулы с коэффициентами, подобранными DeepSeek Web.
3. **Написание кода** — Yandex Sourcecraft подготовлены 4 примера программ на PascalABC.NET (загрузка, описание, базовая модель KNN-регрессии, предобработка).
4. **Подготовка meta-файла и документации** — сформированы meta-файл, README, quality_report, sources.md и данный лог.
