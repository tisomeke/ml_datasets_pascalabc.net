# Источники данных

## mobile_tariff.csv

- **Тип источника:** Полностью синтетические данные
- **Персональные данные:** Не использовались
- **Описание:** Датасет полностью сгенерирован искусственно для задачи регрессии (предсказание цены тарифа мобильной связи). Не содержит реальной информации об абонентах.

### Метод генерации

Данные сгенерированы на основе следующей формулы ценообразования:

$$
\text{price\_rub} = 100 + 7 \cdot \text{internet\_gb} + 0.05 \cdot \text{voice\_minutes} + 0.02 \cdot \text{sms\_count} + 120 \cdot \text{roaming\_flag} + 60 \cdot \text{rollover\_flag} + 80 \cdot \text{messenger\_unlimited\_flag} + 0.03 \cdot \text{internet\_gb}^2 + \varepsilon, \quad \varepsilon \sim \mathcal{N}(0,\,25)
$$

- **Признаки:** internet_gb (1–50 ГБ), voice_minutes (100–2000 мин), sms_count (0–500), roaming_flag, rollover_flag, messenger_unlimited_flag
- **Целевая переменная:** price_rub (рубли)
- **Объём:** 100 строк

### Обоснование коэффициентов

Коэффициенты формулы подобраны на основе анализа реальных тарифов операторов Yota, Билайн, МТС, Мегафон (2025–2026 гг.) с помощью DeepSeek Web. Структура формулы (линейная модель с квадратичным членом за интернет-трафик) отражает типичную систему ценообразования операторов: прогрессивная шкала за объём данных, фиксированные надбавки за дополнительные опции.
